Solve the tetracube

Problem: Given a 6x6x6 empty cube and 54 t-tetracube blocks, the goal is to place the blocks within the empty spaces to form a perfect cube, with no inner spaces or outer protrusions.

This solver will attempt to brute-force a solution by:

  1. Generating batches of randomized blocks (the positions and rotation values are randomized).
  2. Rank these blocks based on highest surface contact if placed into the cube.
  3. Place the highest ranked block into the cube.
  4. Validate that the placement did not create enclosed spaces within the cube.
  5. Remove the last invalid placement, otherwise continue by creating a new batch of block candidates.
  6. Repeat steps 1-5 over a maximum number of retries.
  7. If no blocks can be placed after retry limit, remove some of the lowest ranked surface-touching blocks then continue from step 1.
